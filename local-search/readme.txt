Требуется установка ExifTool, мощного средства для редактирования метаинформации файлов.
Страница проекта: https://www.sno.phy.queensu.ca/~phil/exiftool/
Страница документации к функциям: https://www.sno.phy.queensu.ca/~phil/exiftool/ExifTool.html


interface.sh: получает команды от пользователя, выполняет валидацию данных.
Слово для поиска: Можно указать одно единственное слово для поиска, оно должно быть русским, без пробелов и спецсимволов. При использовании недопустимых слов будет выдано сообщение об ошибке.
Директория: можно указать любую директорию в системе, в ней будет выполняться поиск аудиофайлов и их обработка. По умолчанию принимается текущая директория скрипта.

Использование:
./interface.sh /home/user/ волна
./interface.sh волна

get_meta_from_file.pl: выполняет поиск файлов определённого формата в директории, проверяя метаинформацию файлов с помощью ExifTool. По умолчанию это аудиоформаты WAV и OPUS.

Использование:
./get_meta_from_file /home/user/
